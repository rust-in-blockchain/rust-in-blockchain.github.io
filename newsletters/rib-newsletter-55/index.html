Project Spotlight

Each month we like to shine a light on a notable Rust blockchain project. This month that project is…

iroh.

iroh is a protocol for syncing and moving bytes over peer-to-peer QUIC connections. Data is verified in transit with blake3 verified streaming via the bao-tree crate.

sendme is a simple file transfer program demonstrating the use of iroh.
